# GDD — 快递员：最后一公里

## 1. 概述
- 类型：轻度策略 + 路线规划 + 休闲模拟
- 平台：安卓
- 风格：写实偏卡通、城市日常感
- 单局时长：3–5 分钟
- 核心卖点：路线规划 + 真实事件代入感

## 2. 核心玩法
**一句话**：在有限时间内规划最优配送路线，送完更多订单并维持满意度。

### 2.1 核心循环
接单 → 规划路线 → 出发配送 → 事件干扰 → 结算评分 → 升级

### 2.2 关键决策
- 先送时效高还是距离近？
- 走近路但更堵，还是远路但稳定？
- 资源（时间/体力/道具）该用在哪一单？

## 3. 主要系统

### 3.1 订单系统
- 订单属性：距离、时效、重量、客户等级
- 特殊订单：生鲜/保价/夜间/大件
- 奖惩：超时扣分、投诉率上升

### 3.2 路线规划系统
- 地图节点式路线（简化 TSP）
- 用户拖拽排序送货顺序
- 路线效率评分影响最终收益

### 3.3 事件系统（随机）
- 暴雨：速度下降
- 门禁：等待时间
- 交通管制：强制绕行
- 客户不在：二次派送

### 3.4 评分系统
- 准时率、路线效率、满意度、投诉率
- exact_match（全对）仅用于挑战关卡

### 3.5 升级系统
- 车辆：速度/耐久
- 装备：保温箱、防雨衣、智能导航
- 技能：客户安抚、快速找路、延时补偿

### 3.6 经济系统
- 金币：日常升级
- 代币：稀有装备/皮肤

## 4. UI 页面
- 主界面：城市地图 + 今日订单
- 规划页：订单拖拽排序 + 路线预估
- 配送中：小地图 + 事件弹窗
- 结算页：评分 + 奖励
- 升级页：车辆/装备/技能

## 5. 变现
- 激励广告：免罚一次、收益翻倍、一次重送
- 内购：去广告、车辆皮肤、装备礼包

## 6. 美术风格
- 城市暖色调、生活气
- 角色卡通比例但写实材质

## 7. 风险点与对策
- 路线规划过复杂 → 节点简化 + 推荐路线
- 事件挫败感高 → 提供“道具免罚”
