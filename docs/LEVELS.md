# 关卡与难度曲线

## 城市分区
1. 住宅区（入门）
2. 商业区（中等）
3. 工业区（中高）
4. 老城区（复杂路网）
5. 新区（交通管制多）

## 难度维度
- 订单数量：3 → 8
- 事件频率：低 → 高
- 订单复杂度：普通 → 特殊
- 时间压力：宽松 → 紧张
